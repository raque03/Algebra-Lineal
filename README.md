## Contenido del repositorio

Este repositorio ofrece una exposición rigurosa del álgebra lineal de dimensión finita. Cubre desde la geometría básica de vectores hasta conceptos avanzados como subespacios fundamentales, eigenvalores y el simplex estándar. 

Se integra el rigor matemático con herramientas de programación para:
- Resolución de sistemas de ecuaciones mediante operaciones elementales.
- Manipulación simbólica de expresiones matemáticas.
- Visualización geométrica de proyecciones y transformaciones.
- Análisis de subespacios (Col, Nul, Rango y Nulidad).

## Contenido teórico

### 📌 Espacios y Subespacios
Análisis de estructuras intrínsecas como el Espacio Nulo ($\text{Nul } A$) y el Espacio Columna ($\text{Col } A$), y cómo estos definen la solución de sistemas lineales.

---

### 📌 Transformaciones Lineales
Estudio de operadores que preservan la estructura del espacio vectorial, representados mediante matrices en distintas bases.

---

### 📌 Geometría y Proyección
Uso del producto interno para definir distancias, ángulos y la proyección ortogonal de vectores sobre subespacios, fundamental para problemas de optimización.

---

### 📌 El Simplex Estándar
Conexión entre el álgebra lineal y la toma de decisiones, representando variables como participaciones o probabilidades en un dominio geométrico restringido.

---

## Enfoque del repositorio

El repositorio está diseñado para:
- Proporcionar una base matemática sólida para el área de **Ciencia de Datos** y **Machine Learning**.
- Automatizar cálculos algebraicos complejos mediante **SymPy**.
- Facilitar el aprendizaje de la relación entre matrices y su representación visual.
- Servir como material de referencia rápido para descomposiciones y teoremas fundamentales.
